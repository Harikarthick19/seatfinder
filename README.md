Smart Exam Hall & Seat Locator for Students

SVCE Seat Finder is a lightweight, fast, and fully client-side web app that allows students to instantly find their exam hall, classroom, and seat number by simply uploading the seating-arrangement PDFs and entering their Register Number.

Built for real exam-time usage, with support for all exam sessions (FN / AN / Practical / Retest), and optimized for mobile usage.
